# 🦊 Instructions simples pour Firefox

## 🎯 Étapes rapides

1. **Ouvrez** : https://mapevent.world dans Firefox

2. **Appuyez sur** : `F12` (ou `Ctrl+Shift+K`)

3. **La console s'ouvre en bas** → Cliquez dans la zone de texte (là où il y a `>`)

4. **Tapez** :
   ```
   currentUser.profilePhoto
   ```

5. **Appuyez sur Entrée**

6. **Dites-moi ce qui s'affiche** :
   - Une URL (ex: `https://mapevent-avatars.s3...`) → ✅
   - `null` ou `undefined` → ❌ L'URL n'est pas sauvegardée
