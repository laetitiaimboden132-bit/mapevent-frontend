# Modifications demandées par l'utilisateur

## 1. Vérification d'email après inscription
- Après avoir rempli le formulaire d'inscription, l'utilisateur doit confirmer son adresse email
- Afficher un modal avec un code à 6 chiffres à saisir
- Le code est envoyé par email

## 2. Bloc compte avec nom
- Le bloc compte doit afficher le nom de l'utilisateur (déjà fait via getUserDisplayName)
- Quand on clique sur le bloc, toutes les infos persos apparaissent (agenda, amis, etc.)
- Seulement des blocs, rien d'affiché sauf si on clique sur un bloc

## 3. Option "Rester connecté"
- Dans le formulaire de connexion (Google ou autre), ajouter une checkbox "Rester connecté pour la prochaine fois"
- Gérer la vérification pour les utilisateurs qui restent connectés (comme les leaders mondiaux)

## 4. Modal compte avec blocs
- Créer un modal qui s'ouvre quand on clique sur le bloc compte
- Afficher des blocs : Agenda, Amis, Paramètres, etc.
- Rien n'est affiché par défaut, seulement si on clique sur un bloc
